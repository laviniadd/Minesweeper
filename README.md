## Quick start
1. Clone the repository: 
```bash
git clone https://github.com/laviniadd/Minesweeper.git
```
2. Change directory to repository:
```bash
cd Minesweeper
```
3. Use npm to install the dependencies:
```bash
npm install
```
4. Start the server:
```bash
ng serve
```
5. Navigate to http://localhost:4200/ in your browser.
